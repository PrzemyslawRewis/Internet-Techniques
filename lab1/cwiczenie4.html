Opracować prosty dokument zawierający nawigację wewnętrzną. W ramach skryptu należy wykorzystać co najmniej następujące elementy <p>, <span>, <fieldset>, <legend> i <table>. Do formy prezentacyjnej należy wykorzystać style CSS umieszczone w nagłówku.

  1. Element <p>
  2. Element <span>
  3. Elementy <fieldset> i <legend>
  4. Element <table>

Element <p> tworzy paragraf wewnątrz dokumentu.

Element <span> jest podstawowym wbudowanym kontenerem dla frazowania treści, który z natury nie reprezentuje niczego. Może być używany do grupowania tekstu w celu stylizacji (używa atrybutu class lub id), lub z powodu posiadania takich samych atrybutów, takich jak lang. Powinien być używany tylko wtedy, gdy nie ma zastosowania żaden inny element semantyczny. span jest bardzo podobny do elementu <div>, lecz <div> jest elementem blokowym a <span> elementem liniowym.

Legenda(<legend>) - podpis obramowania

Polecenie <fieldset> pozwala pogrupować tematycznie pewne elementy - wprowadza obramowanie. Zawiera zwykle legendę <legend>. Najczęściej stosowane jest razem z formularzami (<form>).
